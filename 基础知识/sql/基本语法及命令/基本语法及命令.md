## 一些最重要的 SQL 命令
 

- **SELECT** - 从数据库中提取数据
- **UPDATE** - 更新数据库中的数据
- **DELETE** - 从数据库中删除数据
- **INSERT INTO** - 向数据库中插入新数据
- **CREATE DATABASE** - 创建新数据库
- **ALTER DATABASE** - 修改数据库
- **CREATE TABLE** - 创建新表
- **ALTER TABLE** - 变更（改变）数据库表
- **DROP TABLE** - 删除表
- **CREATE INDEX** - 创建索引（搜索键）
- **DROP INDEX** - 删除索引 1.select   用于从数据库中查询数据  
若无条件约束则要使用 *（通配符），默认选择所有的列
基本格式：SELECT column_name(s)
FROM table_name
WHERE condition
ORDER BY column_name [ASC|DESC]

- `column_name(s)`: 要查询的列。
- `table_name`: 要查询的表。
- `condition`: 查询条件（可选）。
- `ORDER BY`: 排序方式，`ASC` 表示升序，`DESC` 表示降序（可选）2.INSERT INTO：用于向数据库表中插入新数据  

```plain
INSERT INTO table_name (column1, column2, ...)
VALUES (value1, value2, ...)
```

- `table_name`: 要插入数据的表。
- `column1, column2, ...`: 要插入数据的列。
- `value1, value2, ...`: 对应列的值。
-  id 列是自动更新的，表中的每条记录都有一个唯一的数字。  3.UPDATE：用于更新数据库表中的现有数据。

```plain
UPDATE table_name
SET column1 = value1, column2 = value2, ...
WHERE condition
```

- `table_name`: 要更新数据的表。
- `column1 = value1, column2 = value2, ...`: 要更新的列及其新值。
- `condition`: 更新条件，哪些数据要修改4.DELETE：用于从数据库表中删除数据。

```plain
DELETE FROM table_name
WHERE condition
```

- `table_name`: 要删除数据的表。
- `condition`: 删除条件。5.CREATE TABLE：用于创建新的数据库表。

```plain
CREATE TABLE table_name (
    column1 data_type constraint,
    column2 data_type constraint,
    ...
)
```

- `table_name`: 要创建的表名。
- `column1, column2, ...`: 表的列。
- `data_type`: 列的数据类型（如 `INT`、`VARCHAR` 等）。
- `constraint`: 列的约束（如 `PRIMARY KEY`、`NOT NULL` 等）。6.ALTER TABLE：用于修改现有数据库表的结构。

```plain
ALTER TABLE table_name
ADD column_name data_type
```

- `table_name`: 要修改的表。
- `column_name`: 要添加的列。
- `data_type`: 列的数据类型。或：

```plain
ALTER TABLE table_name
DROP COLUMN column_name
```

- `column_name`: 要删除的列。7.DROP TABLE：用于删除数据库表。
     DROP TABLE table_name

- `table_name`: 要删除的表。8.CREATE INDEX：用于创建索引，以加快查询速度。

```plain
CREATE INDEX index_name
ON table_name (column_name)
```

- `index_name`: 索引的名称。
- `column_name`: 要索引的列。9.DROP INDEX：用于删除索引。

```plain
DROP INDEX index_name
ON table_name
```

- `index_name`: 要删除的索引名称。
- `table_name`: 索引所在的表。10.WHERE：用于指定筛选条件。
文本字段条件使用单引号‘’ ，数值字段不需使用引号

```plain
SELECT column_name(s)
FROM table_name
WHERE condition
```

- `condition`: 筛选条件。11.ORDER BY：用于对结果集进行排序。

```plain
SELECT column_name(s)
FROM table_name
ORDER BY column_name [ASC|DESC]
```

- `column_name`: 用于排序的列，可以有多个列
- `ASC`: 升序（默认）。
- `DESC`: 降序。12.GROUP BY：用于将结果集按一列或多列进行分组。

```plain
SELECT column_name(s), aggregate_function(column_name)
FROM table_name
WHERE condition
GROUP BY column_name(s)
```

- `aggregate_function`: 聚合函数（如 COUNT、SUM、AVG 等）。13.HAVING：用于对分组后的结果集进行筛选。

```plain
SELECT column_name(s), aggregate_function(column_name)
FROM table_name
GROUP BY column_name(s)
HAVING condition
```

- `condition`: 筛选条件。14.JOIN：用于将两个或多个表的记录结合起来。

```plain
SELECT column_name(s)
FROM table_name1
JOIN table_name2
ON table_name1.column_name = table_name2.column_name
```

- `JOIN`: 可以是 INNER JOIN、LEFT JOIN、RIGHT JOIN 或 FULL JOIN。15.DISTINCT：用于返回唯一不同的值。
仅仅返回某一列中的不同的值

```plain
SELECT DISTINCT column_name(s)
FROM table_name
```

- `column_name(s)`: 要查询的列。
