1.文件读取 readfile函数
readfile ( string: $文件名)
功能：传入一个文件路径，输出一个文件。
2.文件打开  file_get_contents  函数
<?php
   $filename = 'NoAlike.txt';
   $filestring = file_get_contents($filename);
   echo $filestring;
?>
3.修改文件内容
file_put_contents ( string $文件路径, string $写入数据)
功能：向指定的文件当中写入一个字符串，如果文件不存在则创建文件。返回的是写入的字节长度
