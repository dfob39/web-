 有八个魔术常量它们的值随着它们在代码中的位置改变而改变  
1.__LINE__

```plain
<?php
echo '这是第 " '  . __LINE__ . ' " 行';
?>
这是第 “ 2 ” 行

```
2.__FILE__
文件所在路径

```plain
<?php
echo '该文件位于 " '  . __FILE__ . ' " ';
?>
该文件位于 “ E:\wamp\www\test\index.php ”
#绝对路径
```
3.__DIR__
文件所在目录

```plain
<?php
echo '
该文件位于 " '  . __DIR__ . ' " ';
?>
该文件位于 “ E:\wamp\www\test ”
```
4。__FUNCTION__
   返回函数名称
5.__CLASS__
     返回类名

```plain
<?php
class test {
    function _print() {
        echo '类名为：'  . __CLASS__ . "<br>";
        echo  '函数名为：' . __FUNCTION__ ;
    }
}
$t = new test();
$t->_print();
?>  
类名为：test
函数名为：_print
```
6**.__METHOD__**​
** 类的方法名（PHP 5.0.0 新加）。返回该方法被定义时的名字（区分大小写）  **

```plain
<?php
function test() {
    echo  '函数名为：' . __METHOD__ ;
}
test();
?>
函数名为：test
```
7.__NAMESPACE__
 当前命名空间的名称（区分大小写）  

```plain
<?php
namespace MyProject;
 
echo '命名空间为："', __NAMESPACE__, '"'; // 输出 "MyProject"
？>
命名空间为："MyProject"
```
