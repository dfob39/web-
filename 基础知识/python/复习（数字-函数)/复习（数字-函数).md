- **int(x)** 将x转换为一个整数。
- **float(x)** 将x转换到一个浮点数。
- **complex(x)** 将x转换到一个复数，实数部分为 x，虚数部分为 0。
- **complex(x, y)** 将 x 和 y 转换到一个复数，实数部分为 x，虚数部分为 y。x 和 y 是数字表达式。2.字符串

6.字典  用大括号以键值对构成的数据，对之间用逗号分隔
 值可以取任何数据类型，但键必须是不可变的，如字符串，数字  ，元组
通过访问字典里的键来访问字典的值  print("tinydict['Name']: ", tinydict['Name'])
一样可以用del删除某个键值对或者整个字典

9.函数
用def定义
 格式：def 函数名（参数列表）:
             函数体
函数参数之不定长参数   
 1.加了星号 * 的参数会以元组(tuple)的形式导入，存放所有未命名的变量参数  
def printinfo( arg1, *vartuple ):
   "打印任何传入的参数"
   print ("输出: ")
   print (arg1)
   print (vartuple)
# 调用printinfo 函数
printinfo( 70, 60, 50 )
输出: 
70
(60, 50)
​
2. 加了两个星号 ** 的参数会以字典的形式导入 
**可写函数说明**
def printinfo( arg1, **vardict ):   "打印任何传入的参数"   print ("输出: ")   print (arg1)   print (vardict)
**调用printinfo 函数**
printinfo(1, a=2,b=3)
输出: 
1
{'a': 2, 'b': 3}
 如果单独出现星号 *，则星号 * 后的参数必须用关键字传入  

```plain
def f(a,b,*,c):
...     return a+b+c
f(1,2,3)   # 报错
f(1,2,c=3) # 正常
```
