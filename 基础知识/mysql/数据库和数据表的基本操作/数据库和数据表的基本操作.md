二、数据表基本操作
1.创建数据表 创建之前必须指定在哪个数据库中进行相关操作

```html
 create table 表名(
         字段1 字段类型,
         字段2 字段类型,
         …
         字段n 字段类型
);
```

1. show tables;  查看当前数据库中所有表 
2.  show create table student;  查表的基本信息
3. desc student; 查看表的字段信息 
4. **set names utf8;** 命令用于设置使用的字符集  5.修改数据表
alter table student rename to stu;  修改表名
alter table stu change name（旧的字段名） sname（新的字段名） varchar(10); 修改字段名
alter table stu modify sname int;  修改字段数据类型
alter table stu add address varchar(50); 增加字段
alter table stu drop address; 删除字段
6.删除数据表
drop table +表名;
